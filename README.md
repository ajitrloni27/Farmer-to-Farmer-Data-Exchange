🌾 Farmer System – Community Web Application
📌 Project Overview

The Farmer System is a front-end, community-based web application designed to help farmers connect, share knowledge, and interact with each other.

Users can register, log in, create crop-related posts, and engage with others through likes, dislikes, and comments. The platform also allows users to manage their profiles.

The application supports English and Kannada, making it accessible to a wider rural audience.

⚠️ This project is built using HTML, CSS, and JavaScript only, without any backend or database.

✨ Key Features
🔐 User Registration and Login (frontend-based)
🔁 Forgot / Reset Password functionality
🏠 Community Feed to view posts
✍️ Create posts (crop name, description, image)
👍 Like and 👎 Dislike posts
💬 Comment system (comments always visible)
🗑 Users can delete their own posts
👥 View all registered farmers
👤 Profile management (edit name and password)
🌐 Multi-language support (English & Kannada)
📱 Responsive and user-friendly design
🛠 Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and responsiveness
JavaScript (Vanilla JS) – Logic and interactivity
